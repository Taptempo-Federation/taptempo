Tap Tempo
=========

A command line tap tempo. Hit enter key with style and you'll get the corresponding number of beats per minute (bpm).

## Build instructions

### Requirements:
- CMake
- Gettext
- Intl

### Basic build intructions:
```bash
mkdir _build
cd _build
cmake ..
make
install
./taptempo
```
